On the following link you can access the swagger Documentation of the RestFUL API:

https://insp.youpers.com:5000

This is documentation is automatically generated using a tool called Swagger. It lists the supported resources (aka "Classes) and for each resource the supported calls. It also documents the data model that is used in each call, as parameters or in the result.

For simple calls it allows to call the API with test data and look at the returned result directly in the Browser.

Recommended Tools

  • Command Line Tool to call API: HTTPie: http://httpie.org or as an alternative: curl http://curl.haxx.se/
  • Some IDE Plugin to call restful APIs, exists for Eclipse and Intellij IDEA/Webstorm
  • Chrome Browser Plugins for testing Restful APIs are also quite useful
